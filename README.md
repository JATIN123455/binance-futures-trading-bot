# Binance Futures Testnet Trading Bot

## Overview
A simplified trading bot built using Python and the official Binance Futures Testnet API.
Supports Market, Limit, and Stop-Limit orders with proper logging and error handling.

## Features
- Binance USDT-M Futures Testnet support
- Market & Limit orders
- Buy/Sell order sides
- Stop-Limit order (Bonus)
- CLI-based input validation
- Structured, reusable code
- Logging of API requests, responses, and errors

## Tech Stack
- Python
- python-binance
- Binance Futures REST API
